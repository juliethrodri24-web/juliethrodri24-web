## 📞 [Telecommunications Revenue Analysis Megaline](https://github.com/juliethrodri24-web/proyecto_analisis_de_tarifas_por_plan.git)
* **Description:** Comparative statistical analysis between the Surf and Ultimate plans to identify which generates greater profitability and optimize the marketing budget.

* **Tools:** Python, Pandas, SciPy (T-tests), Matplotlib

* **Statistical Approach:** I performed hypothesis tests (T-tests) to validate significant differences in average revenue per user and per region.

* **Key Insight:** The Surf plan generates significant revenue through overage charges, while Ultimate users maintain more stable consumption within their plan.

* **Impact:** Data-driven recommendation for reallocating advertising spending toward the more profitable plan.
